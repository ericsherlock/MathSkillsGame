# MathSkillsGame

//This is a simple math skills game

RULES:

--> 5 questions per round: 
	Addition
	Subtraction
	Multiplication
	Division
	Mod

--> Each Correct/Incorrect Answer Is Awarded A Point Value
	Correct = 10
	Incorrect = -5

--> Once You Reach Certain Point Totals, Your Level Will Increase

--> Reach A Million Points To Win!
